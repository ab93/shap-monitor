# Getting Started

Welcome to shap-monitor! This guide will help you get up and running quickly.

## What is shap-monitor?

shap-monitor is a production ML explainability toolkit designed to monitor SHAP values over time. It helps you track how your model's explanations evolve and detect changes in feature importance patterns.

## Prerequisites

- Python 3.11 or higher
- Basic familiarity with SHAP (SHapley Additive exPlanations)
- A trained ML model
